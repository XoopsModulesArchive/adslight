##  installation de l'URL Rewriting  ##1 - Français2 - English3 - Español########## Français ##########1/ À la racine de votre site, dans votre fichier .htaccessAjouter ces 3 lignes :RewriteEngine onRewriteRule ^annonces/([a-z]+)([0-9]+)(-(.+))? modules/adslight/seo.php?seoOp=$1&seoArg=$2&seoOther=$4&%{QUERY_STRING} [L]RewriteRule ^annonces(/)? modules/adslight/?%{QUERY_STRING} [L]2/ à la racine de votre site, dans un fichier 'robots.txtPlacez-y ces 3 lignes : Disallow: /modules/adslight/Disallow: /modules/adslight/*$Disallow: /modules/adslight/*.php$Ceci afin d'éviter les pages doublons dans les moteurs de recherches. ( Duplicate content )Sinon vous allez perdre en référencement !!...3/ Si votre site est dans un dossier. ex: www.welcome.com/xoops_site/   dans le fichier seo.php   remplacez la ligne comme ceci:   $newUrl = '/yourfile/modules/adslight/' . $seoMap[$seoOp];3/ Activez le ' rewrite url' dans les préférences du module.Votre rewrite url est installer !! Besoin d'aide ? Venez sur le forum : http://www.i-luc.fr/adslight/########## English ##########1/ At the root of your site in your file .htaccessAdd these 3 lines:RewriteEngine onRewriteRule ^annonces/([a-z]+)([0-9]+)(-(.+))? modules/adslight/seo.php?seoOp=$1&seoArg=$2&seoOther=$4&%{QUERY_STRING} [L]RewriteRule ^annonces(/)? modules/adslight/?%{QUERY_STRING} [L]2 / to the root of your site in a file 'robots.txtPut these 3 lines there:Disallow: /modules/adslight/Disallow: /modules/adslight/*$Disallow: /modules/adslight/*.php$This is to avoid duplicate pages in search engines.
